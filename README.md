# CV Application

CV Application for [The Odin Project](https://www.theodinproject.com/).

## Built with

* HTML5
* CSS
* JavaScript
* React
* ESLint

## Features

* Good UI/UX.
* Mobile gestures.
* Responsive design.
* Save resume as image.

You can visit the finished website [here](https://nekusu.github.io/cv-application/).
